# 多模块集成后台

这个项目使用Spring Boot进行开发，尽可能地使用Spring Boot提供的原生方法完成所需功能。目前已完成的功能有：

- 启动时自动更新SQL文件
- 定时任务Schedule的使用
- HTTP请求的使用
- 邮件服务
- MyBatis
- 微信后台
- 博客后台
- 电影记录后台

## 安装方式

项目拷贝下来后，新建一个数据库，名字叫`web`，字符集建议设置为`utf8mb4`，排序方式为`utf8mb4_unicode_ci`。

然后启动程序，运行一遍过后会生成所有的数据表及必要的字段，然后在数据库中找到需要填写的信息补充一下即可。
